# Computer Network Devices Introduction

### Switch

交换机，是一种工作在数据链路层的设备。

以太网是书记链路层最常用的一种网络，而以太网交换机就是以太网的核心设备。

二层交换机的工作流程：

1.  交换机有很多端口，每个端口都有可能从外界交换数据。

    当交换机从某个端口接收到数据包时，先读取包中的源MAC地址，这样就知道源MAC地址对应的是哪个端口；

2.  再读取包中的目的MAC地址，并在地址表中查找相应的端口；

3.  如果表中有这个端口，则把数据包复制到这个端口上；

4.  如果没有，则需要把数据包广播到所有端口上，当目的机器对源机器进行回应时，交换机就可以记录某端口所对应的MAC地址，下次就不需要进行广播了。

交换机的每个端口有一部分缓存，当段时间内有多个报文到达时，需要在缓存中排队等候转发。现代交换机一般还可以在不同端口之间转移缓存，即比较空闲的端口的缓存可以给忙碌的端口用。在缓存中根据优先级队列进行排队。

此外，在交换机中央也有缓存，负责存储要发送到某个端口，但因端口缓存已满还没发送的报文。
